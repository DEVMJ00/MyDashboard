Raccourcis

📄 Office / Productivité

  • Signature Outlook : ALT + JMS
  • Insérer une pièce jointe Outlook : ALT + SAFP
  • Nouvel email Outlook : CTRL + N
  • Répondre à un email : CTRL + R
  • Transférer un email : CTRL + F
  • Rétablir : CTRL + Y

💻 Développement

  • Documentation Python : """Docstring"""
  • Frontmatter YAML : 
    ---
title: ""
description: ""
date: ""
---

    • 🗒️ Conventions de nommage :

    • Classe : CamelCase
    • Constante : SNAKE_CASE
    • Variable / fonction : snake_case
  • Commentaire ligne Python : # commentaire
  • Commentaire HTML : <!-- commentaire -->
  • Commentaire CSS : /* commentaire */

⌨️ Windows - Exécuter (Win + R)

  • Applications utilisateur : %appdata%
  • Données locales : %localappdata%
  • Dossier temporaire : %temp%
  • Version Windows : winver
  • Informations système : msinfo32
  • Configuration système : msconfig
  • Gestionnaire des tâches : taskmgr
  • Gestionnaire de périphériques : devmgmt.msc
  • Gestion des disques : diskmgmt.msc
  • Services Windows : services.msc
  • Éditeur du registre : regedit
  • Programmes installés : appwiz.cpl

🛑 Arrêt / Redémarrage Windows

  • Arrêt immédiat : shutdown /s /t 0
  • Redémarrage immédiat : shutdown /r /t 0
  • Arrêt avec délai de 60 secondes : shutdown /s /t 60
    • Annuler un arrêt programmé : shutdown /a

🌐 Réseau

  • Configuration IP : ipconfig
  • Configuration complète : ipconfig /all
  • Vider le cache DNS : ipconfig /flushdns
  • Tester une connexion : ping adresse_ip
  • Voir les connexions réseau : netstat -ano

🐧 Linux / Raspberry Pi

  • Mettre à jour les paquets : sudo apt update && sudo apt upgrade
  • Voir l'espace disque : df -h
  • Voir les processus : top
  • Voir l'adresse IP : hostname -I
  • Redémarrer : sudo reboot
  • Éteindre : sudo shutdown now
